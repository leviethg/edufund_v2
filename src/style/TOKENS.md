# Design Tokens

## Colors
| Token | Value | Usage |
|-------|-------|-------|
| `primary` | `#0f62fe` | Main Brand Color, Primary Buttons |
| `primary-600` | `#0b5be6` | Hover states |
| `primary-50` | `#eef4ff` | Backgrounds, active states |
| `accent` | `#00b37e` | Success, Progress |
| `accent-warm` | `#ff8a65` | Steps, Highlights |
| `bg` | `#f6f9fc` | Page Background |
| `surface` | `#ffffff` | Cards, Modals |
| `text-main` | `#161616` | Headings |
| `text-secondary` | `#525252` | Body Text |

## Spacing & Radii
| Token | Value | Usage |
|-------|-------|-------|
| `radius-sm` | `4px` | Small elements |
| `radius-md` | `8px` | Inputs, internal containers |
| `radius-lg` | `12px` | Cards, Buttons |
| `radius-xl` | `16px` | Modal, Large sections |

## Typography
Font Family: **Inter**
- Heading 1: Bold, 3-4rem
- Heading 2: Bold, 2.25rem
- Body: Regular, 1rem (16px)
- Small: Medium, 0.875rem (14px)
